Eligibility Criteria

For Undergraduate Programs

  • Completed Class 12 with Physics, Chemistry, and Mathematics
  • Valid national-level entrance examination score
  • Minimum aggregate marks as per current admission brochure

For PG / Ph.D. Programs

  • Relevant bachelor’s/master’s degree from recognized institution
  • Valid GATE / equivalent qualification where applicable
  • Additional interview or written test for selected departments
